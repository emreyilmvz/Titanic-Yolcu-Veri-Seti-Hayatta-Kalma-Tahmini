# Titanic Hayatta Kalma Tahmini 🚢

Bu proje, Kaggle'ın ünlü Titanic veri setini kullanarak yolcuların hayatta kalma durumlarını tahmin eden bir Makine Öğrenmesi çalışmasıdır.

## 📝 Proje Özeti
Bu çalışmada, yolcuların yaş, cinsiyet, bilet sınıfı gibi özelliklerini analiz ederek bir **Karar Ağacı (Decision Tree)** modeli eğitilmiştir.

## 🛠️ Kullanılan Teknolojiler
* **Python 3**
* **Pandas & NumPy:** Veri manipülasyonu ve analizi.
* **Matplotlib & Seaborn:** Veri görselleştirme.
* **Scikit-Learn:** Makine öğrenmesi modeli ve değerlendirme metrikleri.

## 📊 Model Performansı
Modelimiz test verisi üzerinde aşağıdaki başarı kriterlerini yakalamıştır:
- **Doğruluk Oranı (Accuracy):** %81.56
- **Algoritma:** Decision Tree Classifier
